Cloning
-------
To clone the source code, use:
```
git clone https://github.com/youngar/Base9
```

Building
--------

```
make -C ./omr/jitbuilder
cmake .
make
```

To run tests, use:
```
make test
```

To run the benchmark, use:
```
make bench
```
